# 🧮 Operações com Matriz em Java

## 📌 Descrição

Este projeto consiste em uma aplicação em Java que realiza operações básicas com uma matriz quadrada.

O programa lê uma matriz de ordem *n x n*, exibe os elementos da diagonal principal e conta a quantidade de números negativos presentes na matriz.

---

## 🚀 Funcionalidades

* Leitura de uma matriz quadrada (*n x n*)
* Exibição dos elementos da diagonal principal
* Contagem de números negativos na matriz
* Exibição dos resultados

---

## 🧱 Estrutura do Programa

O programa é composto por uma única classe:

* `application1`: responsável pela leitura da matriz, processamento dos dados e exibição dos resultados

---

## 🖥️ Tecnologias Utilizadas

* Java
* Matrizes (arrays bidimensionais)
* Estruturas de repetição `for`
* Estruturas condicionais `if`
* Classe `Scanner` para entrada de dados

---

## 📚 Conceitos Aplicados

* Manipulação de matrizes
* Percurso em estruturas bidimensionais
* Diagonal principal de matriz
* Contagem de elementos com condição
* Entrada e saída de dados
